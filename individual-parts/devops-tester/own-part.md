# DevOps / Тестировщик — Зона ответственности

## Роль
Настройка окружения, тестирование API, проверка работоспособности.

## Задачи

1. **Настройка окружения**
   - Проверить, что проект запускается на чистой машине
   - Документировать процесс установки
   - Помочь команде с настройкой окружения

2. **Тестирование API**
   - Создать коллекцию Postman/Insomnia для всех эндпоинтов
   - Протестировать все CRUD операции
   - Протестировать JWT авторизацию
   - Протестировать soft-delete
   - Протестировать ролевую модель

3. **Проверка качества**
   - Проверить обработку ошибок (400, 401, 403, 404)
   - Проверить валидацию данных
   - Проверить загрузку файлов
   - Проверить историю изменений

4. **Документация**
   - Обновлять README при необходимости
   - Документировать найденные баги
