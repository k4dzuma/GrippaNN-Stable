# Team Lead — Зона ответственности

## Роль
Руководитель проекта. Координация команды, ревью кода, мерж PR, настройка репозитория.

## Задачи

1. **Инициализация репозитория**
   - Создать репозиторий на GitHub
   - Настроить структуру проекта (скопировать общие файлы)
   - Защитить ветку `main` от прямых коммитов
   - Настроить правила PR (require 1 review)

2. **Координация команды**
   - Распределить задачи между участниками
   - Следить за соблюдением Git Policy
   - Помогать с разрешением конфликтов

3. **Code Review**
   - Проверять все PR перед мержем
   - Оставлять комментарии к коду
   - Squash merge одобренных PR в main

4. **Качество проекта**
   - Проверять, что код соответствует ТЗ
   - Убедиться, что soft-delete работает корректно
   - Проверять работоспособность JWT авторизации
